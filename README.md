# iView-project

使用 `@vue/cli` 搭建的 `nodejs` + `mongoDB` + `vue` + `iview` 框架

1. 开发环境下前后端均可实现热更新

2. 生产环境下前端代码可自动打包到服务器端

3. 前后端均使用 `eslint` 规范代码格式

## Install
```bush
// 安装依赖
npm install
```
## Run
### Development
```bush
// 需要打开两个窗口同时执行
npm run serve
npm start
```
### Production(Build)
```bush
npm run build
npm start
```
